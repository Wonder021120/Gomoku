# Gomoku First-Player Advantage

This project investigates first-player advantage in Gomoku under different opening rules, including Standard, Pro, and Swap2.

The project will implement:
- A Gomoku game engine
- Rule variants
- AI agents such as Random, Minimax, MCTS, and NN-MCTS
- Automated AI-vs-AI experiments
- Statistical analysis of first-player win rates

## Environment

The current development environment uses:

- Python 3.10
- NumPy
- Pandas
- Matplotlib
- PyTest
- PyTorch
